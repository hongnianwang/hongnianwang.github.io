The crowd flow prediction  (Sep 2018-present)
Description: Using the public spatio-temporal data (e.g., trajectory or trips data) and deep learning methods to solve prediction problem.
Responsibility: Reading the relevant papers, research model and validate it on public databases, reproduce and contrast with recent methods, and write paper.
Results: The experimental results show our framework is better than the benchmark such as ST-ResNet(AAAI17)， MST3D(ICDM18) and PCRN(IJCAI18) in accuracy and efficiency.
Summary: Spatio-temporal data contains time and space information. For spatial dependencies, we can learn distance and hierarchy features. For temporal dependencies, we need to capture closeness, period and trend features. Therefore, the designed network needs to modeling them both to get better performance.
Paper: Hongnian Wang, Han Su. STAR: A Concise Deep Learning Framework for Citywide Human Mobility Prediction, MDM2019 (CCF C).
